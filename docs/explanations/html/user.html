THÔNG TIN MODULE

Module Name: UserModule

Base Route: /users

File Path: src/user/user.module.ts

Dependencies: PrismaModule, AuthModule (ForwardRef)

Trách nhiệm chính:

  • Quản lý vòng đời người dùng (CRUD).
  • Cung cấp thông tin Profile cho người dùng hiện tại.
  • Tối ưu hiệu năng truy vấn danh sách qua Redis Cache.
  • Đảm bảo an toàn dữ liệu nhạy cảm qua Serialization.
