Giải Thích Chi Tiết Chuyên Sâu - Module Catalog (src/catalog)

Module Catalog là "quầy trưng bày" của hệ thống e-commerce, chịu trách nhiệm quản lý danh mục sản phẩm (Product) và phân loại (Category). Đây là module có tần suất truy vấn cao nhất (Read-heavy).

Lifecycle xử lý request Catalog

Request Lifecycle — GET /catalog/categories
Client
Gửi yêu cầu danh mục
①
CatalogController
Nhận yêu cầu
②
Click vào một node để xem chi tiết.

Các API (CRUD)

GET /catalog/categories findAllCategories()
Input
N/A
Xử lý
Lấy toàn bộ danh mục từ DB
Output
List Categories JSON

Decorators

@Controller('catalog')
Gán prefix /catalog cho tất cả routes trong controller.
