1. Kiến Trúc Toàn Cục (prisma.module.ts)

Module này được đánh dấu bằng decorator @Global():

  • Tái sử dụng liền mạch: Import 1 lần, dùng khắp nơi.
  • Single Source of Truth: Quản lý vòng đời tại một nơi duy nhất.

2. Đi Sâu Vào PrismaService (prisma.service.ts)

PrismaService kế thừa PrismaClient và triển khai OnModuleInit.

2.1. Constructor

constructor(private readonly configService: ConfigService) { ... }

Sử dụng pg Pool và Driver Adapter để tối ưu kết nối, ngăn chặn cạn kiệt kết nối.

2.2. onModuleInit

async onModuleInit() {
  await this.$connect();
}

Ép kết nối ngay lúc khởi động server (Eager Connection), giúp báo lỗi sớm nếu DB cấu hình sai.

3. Luồng Trả Về Dữ Liệu

Kết quả từ PrismaService là dữ liệu thô (Entities), không trả thẳng ra HTTP. Dữ liệu sẽ được xử lý tại Domain Services trước khi ánh xạ sang DTO.
